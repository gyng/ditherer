# Ditherer

Browser-based image/video dithering and glitch art tool. React + Vite + TypeScript + Rust/WASM.

See [AGENTS.md](AGENTS.md) for architecture, component patterns, and coding guidelines.

## Quick Reference

- `npm run dev` — start dev server
- `npm run build` — production build to `build/`
- `npm run test` — run tests
- `npm run lint` — lint (flow + eslint + stylelint)

## Active Plans

- [Modernization Plan](docs/plan/001-modernization.md) — 7-phase migration (bugs, Vite, React 18, TypeScript, Vitest, cleanup, WASM). Check current phase before writing code.
